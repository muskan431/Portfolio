Prasikshan – Employee Training & Registration Management System

Project Summary:

Prasikshan is a comprehensive LAN-based web application developed to streamline the employee registration, training, and course management process within institutional environments like DRDO SSPL. Designed with both functionality and user experience in mind, the system simplifies administrative workflows and ensures seamless handling of training-related operations.

Core Features:

  • Employee Registration Module with personal, departmental, and qualification details
  • Training Form Submission with course selection, completion status, and file upload
  • Admin Portal with functionalities to add, edit, archive, and delete training courses
  • Notice Board for publishing training updates and organizational announcements
  • Print & Preview Functionality for submitted forms and reports
  • Search and Filter Options for quick access to employee records and training logs
  • Role-Based Login System for Admins and Employees
  • Training Completion Verification and certification generation (optional)

Frontend Technologies:

  • HTML5, CSS3, JavaScript for building an interactive user interface
  • Bootstrap for responsive and clean layouts
  • Dynamic Form Validation and status indicators
  • Modal Popups and data tables for better data presentation and interaction

Backend & Server-Side:

  • PHP / Python (Flask) for backend processing (based on stack used)
  • MySQL / SQLite as the database for managing user and training data
  • Session Management for secure access control
  • IIS Server for LAN-based deployment without internet dependency

Design Approach:

  • Focused on usability and clarity for both technical and non-technical users
  • Clean, modular architecture for easy updates and maintenance
  • Dashboard-first design with categorized menus for form handling, notices, and course actions

Impact & Outcome:

Prasikshan digitized the internal employee training workflow, eliminating manual paperwork and reducing administrative effort. It enabled real-time monitoring of training progress and fostered accountability and transparency in employee development. The platform was successfully deployed over a secure LAN environment, ensuring reliability and data control.
